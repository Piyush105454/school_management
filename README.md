# school_management

A comprehensive school management platform with a React frontend and a Django backend.

## Project Structure
- `frontend/`: ReactJS application (Vite-based)
- `backend/`: Django application with modules for admissions, academics, students, attendance, exams, finance, and communication.
- `design/`: Design documents and planning.

## Getting Started
### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`

### Backend
1. `cd backend`
2. Configure your environment and database.
3. `python manage.py runserver`
